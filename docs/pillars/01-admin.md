# Module 1 — Admin

> Business control centre for documents, reminders, tasks, and back-office hygiene.

## 1. Purpose

Admin helps owners stop losing documents in WhatsApp, screenshots, paper folders, and staff chats. It should feel like a simple business command centre, not a corporate document management system.

## 2. Target Users

- Solo entrepreneurs who need one place for business papers.
- Micro SMEs with helpers, receipts, licences, and daily tasks.
- Growing SMEs that need document control before adding approvals, audits, and automation.

## 3. Core Features

Core Admin should cover daily control:

- Digital document storage.
- Business profile document folder.
- Receipt, contract, licence, and HR document folders.
- Secure file sharing link.
- Simple task board: To Do, Doing, Done.
- Task due dates and assignees.
- Basic notification feed.
- Compliance reminder calendar.
- Basic document templates.
- Internal notes.

## 4. Add-on Features

Admin add-ons should carry the premium control layer:

- Extra Storage Pack.
- Digital Signature.
- Custom Document Builder.
- Smart Document Vault.
- Approval Workflow.
- Admin Audit Report.
- Smart Compliance Alerts.
- Admin AI Assistant.

## 5. Key User Flows

### 5.1 Store and share a document

1. Owner uploads a receipt, licence, contract, or HR file.
2. Owner places it in the correct folder.
3. Owner creates a secure share link when someone needs access.
4. The system logs the file and keeps the original private.

### 5.2 Track a daily task

1. Owner adds a task to To Do.
2. Owner assigns it to self or staff.
3. Staff moves it to Doing, then Done.
4. The owner sees completed work in the notification feed.

### 5.3 Remember a compliance date

1. Owner adds an SSM, licence, tenancy, insurance, or permit date.
2. Owner chooses reminder timing.
3. The reminder appears before the deadline.
4. Owner marks done and schedules the next cycle.

## 6. Data Notes

Core tables should cover files, folders, tasks, notifications, compliance reminders, and document template instances. Sensitive HR files should be flagged so access can be limited by role.

Premium add-ons can extend the same records with signature events, approval states, audit exports, categorization labels, storage quotas, and AI-generated summaries.
